# routes-analysis 使用指南
## 🛠️ 安装
### 源码安装
```bash
git clone https://github.com/jascga/routes-analysis.git
cd routes-analysis
pip install -e .
# 验证
routesanalysis --version
```
### 二进制使用
下载发布页的`routesanalysis_vx.y.z_windows_x64.exe`，直接在cmd/终端运行即可。
---
## 📌 常用命令
### 通用参数
```bash
# 查看版本
routesanalysis --version
# 查看帮助
routesanalysis --help
# 查看子命令帮助
routesanalysis <子命令> --help
```
### 场景1：多组平行设备负载分担分析（multi-group）
```bash
# 基础用法，输出Excel报告
routesanalysis multi-group device.txt -o result.xlsx
# 自定义最小命中分组数（默认≥2组命中）
routesanalysis multi-group device.txt -m 3 -o result.xlsx
# 指定文件编码（默认自动识别，支持utf-8/gbk）
routesanalysis multi-group device.txt -e gbk -o result.xlsx
# 只输出终端摘要，不生成Excel
routesanalysis multi-group device.txt --no-excel
# 批量处理多个文件，每个生成一份报告到output目录
routesanalysis multi-group *.txt -o ./output/
```
### 场景2：多设备BGP路由表比较（compare）
```bash
# 基础用法：比较device1.txt和device2.txt，默认以第一个文件为基准
routesanalysis compare device1.txt device2.txt -o diff.xlsx
# 指定第2个文件为基准（索引从0开始）
routesanalysis compare -b 1 device1.txt device2.txt device3.txt -o diff.xlsx
# 指定编码
routesanalysis compare -e gbk device1.txt device2.txt -o diff.xlsx
# 只输出终端差异摘要
routesanalysis compare device1.txt device2.txt --no-excel
```
### 辅助命令：快速查看文件解析结果
```bash
routesanalysis inspect device.txt
```

## 📦 Windows打包
### 环境要求
- Windows 10/11
- Python 3.9+，安装时勾选`Add Python to PATH`
### 打包步骤
1. 克隆项目到本地
2. 进入项目根目录，双击运行`scripts\build.bat`
3. 等待1-3分钟，输出文件在`dist\routesanalysis.exe`

## 📝 输入文件格式要求
### 路由表采集
在华为交换机上执行，保存为文本文件：
```bash
display ip routing-table protocol bgp
```
### 可选：接口描述采集（优化对端设备识别）
同设备执行，和路由表内容合并到同一个文件（顺序无要求）：
```bash
display interface description
```
