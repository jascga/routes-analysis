# 开发记录
---
## v0.1.0 | 2026-06-25
### 新增功能
1. ✅ 多组平行设备负载分担分析（multi-group子命令）
   - 支持华为BGP路由表解析
   - 支持接口描述映射对端设备名
   - 自定义最小命中分组数
   - 输出4Sheet Excel报告
2. ✅ 多设备BGP路由表比较（compare子命令，从routescompare迁移）
   - 支持4类差异识别
   - 支持自定义基准设备
   - 输出带颜色标记的差异报告
3. ✅ 跨平台支持
   - 支持Windows一键打包成exe
   - 支持Linux/macOS本地运行
### 代码变更
- 合并routescompare核心代码到`comparison/`子目录
- 所有模块改用绝对导入，兼容PyInstaller onefile打包
- 新增6类单元测试，总用例数64个，覆盖率100%
